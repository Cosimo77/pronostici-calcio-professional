Informazioni Sistema

Come funziona il sistema di predizioni

Modelli AI

Il sistema utilizza un ensemble di 3 modelli di Machine Learning: Random Forest, Gradient Boosting e Logistic Regression per predizioni accurate.

  • Accuracy ~50%
  • 32 Features engineered
  • Cross-validation 5-fold

Dataset

Analisi basata su dati storici reali della Serie A italiana dal 2020 al 2025, con oltre 1,900 partite analizzate.

  • 1,940 partite
  • 28 squadre
  • 5 stagioni complete

Features

Ogni predizione utilizza oltre 30 variabili tra forma recente, performance casa/trasferta, head-to-head e statistiche avanzate.

  • Forma ultime 5 partite
  • Performance casa/trasferta
  • Scontri diretti
